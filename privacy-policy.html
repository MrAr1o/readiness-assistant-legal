2. What This App Does

Readiness Assistant is a documentation readiness tool that scans Confluence Cloud spaces to help teams:

  • identify evidence gaps against compliance frameworks (GDPR, ISO 27001, SOC 2, and others);
  • detect stale, ownerless, or poorly structured documentation;
  • assign ownership and track remediation workflows;
  • generate internal Readiness Audit Packs.

This app does not certify compliance, provide legal advice, issue audit opinions, or guarantee regulatory approval.

3. Scope of This Policy

This policy covers data processed and stored by Readiness Assistant when installed as an Atlassian Marketplace app on Confluence Cloud. It does not cover:

  • Atlassian's own data practices (governed by Atlassian's Privacy Policy);
  • Confluence Cloud itself;
  • any linked third-party services you may use independently.

5. How We Process Page Content

When a scan is triggered:

  1. Page content is fetched via the Confluence API and loaded into memory within the Atlassian Forge runtime.
  2. Processing occurs entirely within the Forge serverless environment — no data is sent to external servers.
  3. The text is analysed for evidence keywords (compliance scan) and documentation health signals (staleness, ownership).
  4. If the optional Sensitive Data Check feature is enabled, page text is additionally scanned for common credential patterns.
  5. After analysis, the raw page body text is discarded. It is not stored in Forge Storage.

6. What We Store in Forge Storage

All persistent data is stored in Atlassian Forge Storage — a per-installation, isolated key-value store hosted within Atlassian's infrastructure. Each installation has its own isolated storage namespace. No data crosses tenant boundaries.

7. Sensitive Data Handling

The Sensitive Data Check feature is optional and disabled by default. When enabled by an admin:

  1. Page content is scanned for credential patterns (API keys, tokens, connection strings, PII) within the Forge runtime.
  2. When a pattern match is detected, only a redacted preview is created (first and last few characters visible).
  3. The redacted preview — not the original value — is stored in Forge Storage.
  4. The original credential or sensitive value is discarded immediately after detection.

No credential is ever stored in plain text.

The app is built on Atlassian Forge, Atlassian's serverless platform for Confluence Cloud. By using this app, your data is processed within Atlassian's infrastructure subject to Atlassian's own data handling practices.

Atlassian acts as a subprocessor for:

  • Runtime execution of the app's functions;
  • Storage of app data in Forge Storage.

Atlassian's Privacy Policy: https://www.atlassian.com/legal/privacy-policy

12. Customer Admin Controls

Confluence admins who have installed the app can:

  • Clear all findings: Use the "Reset Data" function in the app Settings panel to delete all stored scan findings and scores.
  • Exclude spaces: Configure space exclusion rules to prevent specific spaces from being scanned.
  • Disable optional features: Turn off the Sensitive Data Check feature if not needed.
  • Uninstall the app: Removes all Forge Storage data for that installation.

We do not currently provide a self-service export of all stored data. Admins who need a full data export for compliance purposes may contact support@steiktech.de.

13. Data Residency

Readiness Assistant stores data in Atlassian Forge Storage. Atlassian determines the physical location of Forge Storage based on their infrastructure and any data residency options available to your Atlassian organisation.

We do not independently control data residency. If data residency location is a requirement for your organisation, consult Atlassian's documentation on Forge Storage data residency and your Atlassian subscription agreement.

14. Security

  • The app runs exclusively within the Atlassian Forge sandboxed runtime. It does not run on external servers.
  • Network calls from the app are limited to Atlassian APIs (Confluence API, Forge Storage API). No other outbound connections are made in v1.
  • Sensitive pattern matches are redacted before storage.
  • Forge Storage is isolated per app installation — no other Marketplace app can access your installation's data.

We maintain a responsible disclosure process. To report a security concern, contact: support@steiktech.de
